# cat-dog-ai-project
AI based image classification project that identifies whether an image is a cat or a dog using deep learning

# PROJECT OVERVIEW
This project is based on Artifical Intelligence and Deep Learning.It classifies images into two categories Cat and Dog

# Obj ective

To build a model that can accurately predict whether a given image is a cat or a dog.

# Features

- Image classification using AI
- Data preprocessing
- Model training and testing
- Prediction on new images

# Technologies Used

- Python
- NumPy
- TensorFlow / Keras
- Matplotlib

# How to Run

1. Install required libraries:
   pip install numpy tensorflow matplotlib

2. Run the notebook:
   model.ipynb

# Results

The model predicts whether the image is a cat or a dog with good accuracy.
